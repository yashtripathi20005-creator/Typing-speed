# ⌨️ Typing Speed Test

A modern, feature-rich typing speed test application built with Python and PyQt5.

## Features

- 🎯 **Real-time typing test** with 60-second timer
- 📊 **Detailed statistics** including WPM, accuracy, and character count
- 🎨 **Modern dark theme** with colorful UI
- 📝 **Random word generation** for varied practice
- ✅ **Real-time feedback** with word highlighting
- 🏆 **Performance ratings** based on your results
- ⌨️ **Keyboard shortcuts** (ESC to end test)
